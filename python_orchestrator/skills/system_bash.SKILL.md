Tool Name: system_bash
Description: Ejecuta un comando bash nativo en el sistema operativo del usuario. Úsalo cuando el usuario pida explícitamente crear, borrar o modificar carpetas, archivos, o interactuar con la computadora. NO USES esto para tareas internas, solo para peticiones del usuario.
Arguments:
- command (string): El comando bash exacto a ejecutar (ej. 'mkdir tesis_test', 'ls -la', 'pwd').
